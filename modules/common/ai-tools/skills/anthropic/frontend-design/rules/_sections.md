# Sections

This file defines all sections, their ordering, impact levels, and descriptions.
The section ID (in parentheses) is the filename prefix used to group rules.

---

## 1. Design Thinking (design)

**Impact:** CRITICAL
**Description:** Conceptual direction, purpose, and tone.

## 2. Visual Aesthetics (visual)

**Impact:** HIGH
**Description:** Typography, color, motion, and visual details.

## 3. Composition & Layout (composition)

**Impact:** HIGH
**Description:** Spatial arrangement, grids, and layout patterns.

## 4. Implementation (code)

**Impact:** MEDIUM
**Description:** Technical implementation, code quality, and framework usage.
